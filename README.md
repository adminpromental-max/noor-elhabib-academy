# أكاديمية نور الحبيب

صفحة هبوط لأكاديمية نور الحبيب — تحفيظ القرآن الكريم والتجويد والدراسات الإسلامية.

## المميزات

- تصميم عربي أصيل بخطوط Amiri و Reem Kufi و Cairo
- ألوان مستوحاة من شعار الأكاديمية (أخضر زمردي + ذهبي + كريمي)
- حركات تفاعلية باستخدام GSAP
- 8 برامج تعليمية مع أزرار حجز عبر واتساب
- متجاوب مع جميع الشاشات

## التشغيل محلياً

```bash
npm run dev
```

أو افتح `index.html` مباشرة في المتصفح.
